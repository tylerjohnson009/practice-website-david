Project

Hammer & Tongs Vermouth

Description

Hammer & Tongs is a bold brand with an antique, yet somehow modern feel. I developed a website to match using a custom CMS solution that allows the owner to maintain new products and news updates at their convenience. This project is so new that we’re still gathering bottle shots and a few other pieces of information, but we couldn’t wait to launch to share Hammer & Tongs with the world!

Responsibilities

website design, website development, custom cms

website design, website development, wordpress

Project

Seven of Hearts

Description

Designed a new website with the intention of cleaning up and modernizing the winery’s online identity. I also added a customized e-commerce platform, allowing Seven of Hearts to sell wine to new markets.

Responsibilities

website design, website development, responsive design, wordpress, e-commerce development
